# `@pollux/web`

Next.js management web interface for the Pollux bot framework.

## Scripts

| Script | Description |
|---|---|
| `dev` | Start development server on port 3026 |
| `build` | Build for production |
| `start` | Start production server on port 3026 |
| `lint` | Run ESLint on source files |

## Development

```sh
yarn workspace @pollux/web dev
```

Open [http://localhost:3026](http://localhost:3026) in your browser.
